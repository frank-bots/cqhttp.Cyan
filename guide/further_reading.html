    CQApiClient的参数

    • CQHTTPClient的参数中除access_url外皆为可选参数，即你可以实例化一个“仅发送API请求”的Client对象。
    • 你可以通过指定CQApiClient的Caller与Listener来以不同的连接方式连接到cqhttp(好像没太大必要？)

    多实例

    你可以实例化多个Client对象以连接到不同的酷Q实例，以完成各种骚操作

    using cqhttp.Cyan.Utils

    Utils命名空间中有很多好用的工具

    Name Description Usage
    GroupTable/MessageTable 持久化存储群列表与群员信息GroupTable / 持久化存储两分钟内发送的消息MessageTable 向CQApiClient的构造函数中传入use_group_table: true，则在这个Client示例调用GetGroupList API时将自动向Client的group_table记录获得的群组信息。
    Dialogue 可以用来构建对话系统 参考WeeklyReportBot
    Convert.* 各种用于转换消息用的函数 如Convert.ToEmoji(id)可以将QQ表情转换为相似的Emoji，便于发送奇怪的表情
    Logger 一个简易的日志类，构造方式类似Python中的日志 var logger = Logger.GetLogger("my_logger");
